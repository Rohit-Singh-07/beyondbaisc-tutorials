1. Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating webpages. It uses tags to structure and organize content.

HTML files always begin with a Document Type Declaration (<!DOCTYPE html>). 

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>

2. The <head> Section

The head contains metadata and information that is not directly displayed.

  • <meta> — Defines metadata such as encoding, description, keywords.
  • <title> — Defines the title displayed in the browser tab.
  • <link> — Connects external resources like icons or CSS.
  • <script> — Links JavaScript files or includes inline scripts.

4. Text Formatting Tags

HTML provides various tags to style text meaningfully:

  • Bold: <b> or <strong>
  • Italic: <i> or <em>
  • Underline: <u>
  • Mark/Highlight: <mark>
  • Small Text: <small>
  • Deleted Text: <del>
  • Inserted Text: <ins>
  • Superscript: 2<sup>3</sup> = 23
  • Subscript: H<sub>2</sub>O = H2O

15. Block vs Inline Elements

Block Elements: Start on a new line (e.g., <div>, <p>, <table>)

Inline Elements: Continue on the same line (e.g., <span>, <a>, <img>)
